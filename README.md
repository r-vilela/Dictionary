# Dicionary System

A simples dictionary from Spanish to Portuguese and from Portuguese to Spanish using an AVL tree to structure the words. The system automatically reads and stores the words on a .txt file.

## How to run

Clone this repository with:
`git clone git@github.com:r-vilela/Dictionary.git`

Enter the project folder and run:
`gcc main.c AVL/avl.c Dict/dict.c -IAVL/ -IDict/ -o main`
